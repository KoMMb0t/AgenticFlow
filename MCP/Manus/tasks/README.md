# Manus Doc-Task Inbox

Hier landen automatisch erzeugte **Doc-Update-Tasks** vom Workflow `tools/doc-watch.ps1`.

- `doc-task-<timestamp>.md` — offener Task (von Manus abzuarbeiten)
- `_CHANGELOG.md` — Protokoll erledigter Doku-Updates
- `done/` — abgeschlossene Tasks (von Manus hierher verschoben)

Ablauf & Schreibregeln: siehe `../MANUS-DOC-AGENT.md`.
